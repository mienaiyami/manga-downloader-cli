# Manga Downloader

---

Supported sites:
- https://mangareader.to/  (can't download shuffled images.)
- https://cubari.moe/  (gist link only e.x. https://gist.githubusercontent.com/)

---

## How to use

Require nodejs v14+

#### Installation

Run
``` npm install```

or 

```yarn install```

in base directory.

#### Run app

``` node .```

in base directory.
